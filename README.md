## About

緯度経度を地図上に可視化・分析するためのツールです。  

https://tool.geo.qazsato.com  

下記4種類の表示が可能です。  

1. 住所 (都道府県・市区町村など)
2. 地域メッシュ
3. ヒートマップ
4. マーカークラスタリング
